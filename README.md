# PenExpreSSL 

PenExpreSSL is a very simple Express server with SSL to be used for quick payload delivery.

## Installation

Use the package manager [Yarn](https://google.ch/) to install PenExpreSSL.

```bash
cd PenExpreSSL
yarn install
```
